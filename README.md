## Run Locally

Clone the project

```bash
  git clone https://github.com/nfortino/nico-dev.git
```

Go to the project directory

```bash
  cd nico-dev
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
